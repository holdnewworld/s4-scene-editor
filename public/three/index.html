S4 Scene Editor - Angular and ThreeJS demo

Requires desktop browser -- Chrome, FireFox, or IE11 (IE has issues)

Demo features
  1. AngularJS binding for commands and data input
  2. AngularJS async ajax-like JSON resource retrieval
  3. JSON parsing (Using S4 -- "Steve's Super Simple Scene Format")
  4. Three.JS animation
  5. Live scene editing
Controls Object Rotation Speed

Scene Code Editor (Must be valid JSON)

Try these edits...
  • Try editing the hex values for colors to some other hex value, e.g. 0x236581
  • Replace "Cylinder" with "Cone" and change "Height" from 1 to 2
  • Adding another "Box," "Cylinder," or "Cone" object with position, dimension, and color properties to the end of the JSON array.
