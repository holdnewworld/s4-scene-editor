S4F Scene Editor

Requires desktop browser -- Chrome, FireFox, or IE11. Later Android (4.4, etc..) browsers may be supported. Safari/Safari mobile may be supported with OSX10.10/iOS8.

Demo features

  1. AngularJS binding for commands and data input
  2. AngularJS async ajax-like JSON resource retrieval
  3. JSON parsing (Using S4F -- "Steve's Super Simple Scene Format")
  4. Three.JS models, textures, lights, and animation
  5. Live scene editing--ace editor support coming soon!
